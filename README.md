# :mouse: Darrel the Mouse

Darrel the Mouse is a 2D side scroller video game developed in Unity3D 5.6.  It's mobile friendly!  If you would like to collabrate in anyway, let me know.

### Requirements:

* Facebook Unity SDK 7.9.4
